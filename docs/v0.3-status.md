## R1 highlights (closed)

- C-sec: rate limit on /sage/query, entidad parameterized
- C-perf: SSE cancel threading, Anthropic SDK 30s timeout, openai-compat AbortSignal.timeout, pg_advisory_xact_lock per-thread
- C-coh: SAGE_SQL_SCHEMA_SUMMARY regenerated from pg_attribute (9 of 13 tables had wrong column names), SESNSP XX998/XX999 catch-all exclusion, partial-year filter on delitos, EDR pinned to 2024
- W-sec: redactPgError → opaque codes, max_rows validation, MAX_TURNS_PER_THREAD=50
- W-perf: widened exec gap from +2s → +5s, maxBuffer 16MB → 64MB
- W-coh: farmacias_controlados renamed to farmacias_endorsements_controlados (honest count semantics)
