## What's in the live DB

### Establecimientos (the spine)

`establecimientos` — 6,097,681 rows × wide schema. Backfilled join keys:

- `entidad` (CVE_ENT, 2-char) — 100% populated
- `area_geo` (CVE_MUN_5, 5-char ENT||MUN) — backfilled from CLEE chars 1-5
- `sector_actividad_id` / `subsector_actividad_id` / `rama_actividad_id` /
  `subrama_actividad_id` / `clase_actividad_id` (SCIAN 2/3/4/5/6-digit) —
  backfilled from CLEE chars 6-11
- `ageb` (13-char CVEGEO ENT+MUN+LOC+AGEB) — backfilled via PostGIS spatial
  join against `ageb_polygons`. 99.99975% coverage (15 unmatched are bad
  lat/lon).

Indexes: btree on entidad, area_geo, clase_actividad_id, sector_actividad_id;
GIST on geom; GIN on nombre.

## What blocks v0.2.1 from full closure

**AGEB-level disaggregates**:

- INEGI `RESAGEBURB` (Censo 2020 by AGEB and manzana) — per-state CSVs
  gated behind `inegi.org.mx/app/scitel/Default?ev=10` SCITEL portal.
  UA spoofing alone doesn't crack it. **Operator-assist needed.**
- CONEVAL "Grado de Rezago Social en AGEB urbanas 2020" — same pattern,
  separate gated portal at coneval.org.mx.

If/when an analytical use case demands AGEB demographics, the path is:

1. Operator manually downloads from the CONEVAL/INEGI portal in a browser
2. Hands me the file path or signed URL
3. I extend the loaders to handle the schema (similar shape to ITER)
4. Spatial join against `ageb_polygons.cvegeo`

## Loader scripts (all idempotent, all in `scripts/`)

| Script             | Inputs                          | What it does                                                                       |
| ------------------ | ------------------------------- | ---------------------------------------------------------------------------------- |
| `load-censo.ts`    | `--csv=ITER.csv`                | DROP+CREATE `censo_iter`, `\copy` with NULL '\*', recreate `censo_municipios` view |
| `load-coneval.ts`  | `--pobreza=p.csv --irs=irs.csv` | DROP+CREATE both raw tables, `\copy`, recreate both views                          |
| `backfill-ageb.ts` | (no flags)                      | Spatial-join UPDATE on establecimientos via ageb_polygons                          |
| `pipeline.ts`      | `--all` / `--estados=09`        | Full DENUE national or per-state extraction (prints reminder to run backfill-ageb) |

All hardened with: csvPath leading-`-` rejection, dbContainer regex, `cp --` separator, try/finally cleanup, quoted SQL identifiers, NULLIF guards, mocked orchestration tests.
